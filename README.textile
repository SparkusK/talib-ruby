h2. talib-ruby

Ruby Wrapper for "ta-lib":http://ta-lib.org/

This project has been started by Timur Adigamov on "Rubyforge":http://rubyforge.org/projects/talib-ruby/,
but since it didn't build on my machine and wasn't complete i modified it slightly, et voila.

h3. Install and build instructions

Install ta-lib from "MacPorts":http://www.macports.org/ports.php?by=name&substr=ta-lib:

bc. sudo port install ta-lib

Checkout talib-ruby (it includes the ta-lib directories /include and /lib from the MacPorts version)

Now run the following commands in talib-ruby:

bc. ARCHFLAGS="-arch i386" ruby extconf.rb --with-talib-dir=/ABSOLUTE_PATH_OF/talib-ruby

-> should output 'checking for TA_Initialize() in -lta_lib... yes' and 'creating Makefile'

bc. make

-> creates talib.bundle

bc. sudo make install

-> copies talib.bundle to /Library/Ruby/Site/1.8/universal-darwin9.0

Now ta-lib can be used by using _require 'talib'_
